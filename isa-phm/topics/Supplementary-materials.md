## Dataset 1 - NLN-EMP

This is the dataset from:

_Bruinsma, S., Geertsma, R. D., Loendersloot, R., & Tinga, T. (2024). Motor Current and Vibration Monitoring Dataset for various Faults in an E-motor-driven Centrifugal Pump. Data in Brief, 52, 1-16, Article 109987. [https://doi.org/https://doi.org/10.1016/j.dib.2023.109987](https://doi.org/https://doi.org/10.1016/j.dib.2023.109987)_

Link to the data: [https://doi.org/10.4121/2b61183e-c14f-4131-829b-cc4822c369d0.v4](https://doi.org/10.4121/2b61183e-c14f-4131-829b-cc4822c369d0.v4)

The following files are provided and can be downloaded:

| File name                                         | Description                                               |
|---------------------------------------------------|-----------------------------------------------------------|
| <resource src="Input_template_NLN_EMP.xlsx"/>     | The completed input file that generates the ISA-PHM files |
| <resource src="ISA_Output_isatab_NLN_EMP.xlsx"/>  | The resulting ISA-PHM file in ISAtab format               |
| <resource src="ISA_output_isajson_NLN_EMP.json"/> | The resulting ISA-PHM file in ISA json format             |


Note that the associated raw / processed data files are (still) missing.
